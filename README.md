# mutahar789.github.io

Source for my personal academic homepage: **[mutahar789.github.io](https://mutahar789.github.io)**

Built with [Jekyll](https://jekyllrb.com/) and deployed via GitHub Pages. Most content lives in [`index.md`](index.md); styling is in [`_sass/`](_sass/) and [`assets/css/`](assets/css/).

## Local preview

```bash
bundle install
bundle exec jekyll serve --livereload
```

Then open <http://127.0.0.1:4000>.
